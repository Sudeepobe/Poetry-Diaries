# Poetry Diaries Documentation

## Editing Poems
To edit poems, navigate to the ` poems/` directory and select the poem you wish to edit. You can modify the content in Markdown format. Make sure to keep the structure consistent to maintain the overall style.

## Customizing Styling
You can customize the styling by editing the `styles.css` file in the `assets/` folder. Modify the styles as needed to reflect your personal aesthetics. Make sure to test the styles locally before committing.

## Deploying to GitHub Pages
To deploy your project to GitHub Pages:
1. Ensure your repository settings are configured to serve from the `gh-pages` branch.
2. Run the deployment script included in the root of the project:
   ```bash
   npm run deploy
   ```
3. Check the `settings` tab in your GitHub repository to ensure GitHub Pages is enabled and linked to the correct branch.

For more information, refer to the [GitHub Pages documentation](https://pages.github.com/).
